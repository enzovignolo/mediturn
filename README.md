# mediturn
App for manage appointments for doctors


# v1:

## Endpoints:

### /v1/patients

  -GET all patients registered
  -POST add patient to the DB

### /v1/patients/:id

  -GET patient by Mongo object id
  -PUT update patient information by Mongo object id
  -DELETE patient by Mongo object id

### /v1/doctors

  -GET all doctors registered
  -POST add doctor to the DB

### /v1/doctors/:id

  -GET doctor by Mongo object id
  -PUT update doctor information by Mongo object id
  -DELETE doctor by Mongo object id
